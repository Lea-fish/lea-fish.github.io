Leafish

Een open-source Minecraft client geschreven in RUST

Open-source

Leafish is een community project en compleet open-source. Geen originele Mojang code wordt gebruikt

Geschreven in Rust

Dit betekent moderne platform ondersteuning en prestatie verbeteringen niet mogelijk in het oorspronkelijke spel

Multi-versie ondersteuning

Leafish is van de grond af opgebouwd om meerdere versies tegelijk te ondersteunen. Er is geen noodzaak om tussen meerdere versies te wisselen, deze client can elke server van 1.7.10 en verder joinen
